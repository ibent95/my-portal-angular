Dynamic forms service in Laravel Lumen

Layanan ini merupakan REST Service / API yang khusus menangani kebutuhan formulir dinamis (Dynamic Form). Menggunakan bahasa pemrograman PHP serta framework Laravel Lumen.

Tujuan pembuatan layanan ini untuk memfasilitasi pengembang (pribadi ataupun orang lain) dalam proses pengembangan antarmuka (User Interface) formulir dinamis. Kebutuhan pengembangan tersebut meliputi :

  • menampilkan jenis formulir
  • menampilkan daftar pengaturan formulir sesuai dengan jenis formulir
  • menyimpan data sesuai dengan jenis formulir
